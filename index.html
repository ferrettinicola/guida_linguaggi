01

Linguaggi del Web Frontend

Browser

Tutto ciò che vedi in un sito web è costruito con questi tre linguaggi. Girano direttamente nel browser dell'utente, senza server.

HTML markup

Non è un vero linguaggio di programmazione: è un linguaggio di struttura. Definisce cosa c'è nella pagina — titoli, paragrafi, bottoni, immagini. È come lo scheletro di un corpo.

<h1>Ciao mondo</h1> <button>Cliccami</button> <img src="foto.jpg">
Siti web Email HTML App web
CSS stile

Definisce come appare tutto ciò che HTML ha strutturato. Colori, font, dimensioni, animazioni, layout. Quello che hai visto nelle immagini Instagram era principalmente CSS. È come la pelle e i vestiti del corpo.

.bottone { background: #4ade80; border-radius: 8px; transition: all 0.3s ease; }
Design web Animazioni Layout
JavaScript linguaggio

L'unico vero linguaggio di programmazione che gira nel browser. Gestisce la logica interattiva: cosa succede quando clicchi un bottone, le animazioni dinamiche, le richieste a internet. La lampada dell'immagine 2 era tutta JavaScript. È il cervello.

function accendiLampada() { if (isOn) { body.style.background = "#1c1f24"; } }
Interattività Animazioni App web
💡 Analogia: HTML è lo scheletro, CSS sono pelle e vestiti, JavaScript è il cervello che fa muovere tutto.
02

Linguaggi per Dati & Calcolo

Scientifico

Linguaggi pensati per analisi statistica, calcolo numerico e manipolazione di dati. Girano sul tuo computer o su un server, non nel browser.

Python linguaggio

Il linguaggio più versatile che esiste oggi. Nato generico, è diventato dominante in data science, AI e machine learning grazie alle sue librerie. Sintassi semplice e leggibile. Quello che già usi tu.

import pandas as pd df = pd.read_csv("dati.csv") media = df["colonna"].mean() print(media)
Data science AI / ML Automazione Web backend
R linguaggio

Linguaggio nato specificatamente per la statistica. È il riferimento nel mondo accademico, biologia, medicina ed economia. Ha funzioni statistiche avanzate native che in Python richiedono librerie. Meno versatile di Python ma superiore nella statistica pura.

# Analisi statistica semplice dati <- c(12, 15, 18, 22, 30) mean(dati) # media sd(dati) # deviazione standard plot(dati) # grafico immediato
Statistica Ricerca Grafici Biostatistica
MATLAB linguaggio + software

È sia un linguaggio che un ambiente tutto-in-uno, sviluppato da MathWorks. Eccelle nel calcolo matriciale, simulazioni fisiche e ingegneristiche. È proprietario e costoso (licenze universitarie). Molto usato in ingegneria, fisica e telecomunicazioni. Il codice non è portabile come Python o R.

% Risoluzione sistema lineare A = [1 2; 3 4]; b = [5; 6]; x = A \ b; % soluzione automatica plot(x)
Ingegneria Fisica Simulazioni Segnali
⚠️ Nota: MATLAB ha un'alternativa gratuita chiamata Octave, quasi identica nella sintassi.
03

Ambienti di Sviluppo

Software

Non sono linguaggi. Sono i programmi che usi per scrivere codice. La stessa differenza tra Word (software) e l'italiano (linguaggio).

RStudio IDE per R

È semplicemente il miglior editor per scrivere R. Divide lo schermo in pannelli: codice, output, grafici, variabili. Senza RStudio potresti usare R ugualmente, ma sarebbe molto meno comodo. È gratuito.

Editor di R Visualizzazione grafici Console interattiva
Jupyter Notebook IDE per Python

Ambiente interattivo per Python che gira nel browser. Permette di mescolare codice, testo e grafici in un unico documento. Molto usato in data science e nei corsi universitari.

Python interattivo Data science Didattica
VS Code IDE universale

L'editor più popolare al mondo, gratuito di Microsoft. Supporta qualsiasi linguaggio tramite estensioni: Python, R, JavaScript, HTML, CSS. Probabilmente l'editor che usano i creator delle immagini Instagram che hai visto.

Tutti i linguaggi Gratuito Estensioni infinite
💡 Analogia: RStudio sta a R come Word sta all'italiano. Il linguaggio esiste indipendentemente dal software che usi per scriverlo.
04

Linguaggi di Formattazione

Documenti

Non programmano logica. Descrivono come appare un documento. Niente variabili, niente algoritmi — solo struttura e stile testuale.

LaTeX markup scientifico

Il sistema standard per scrivere documenti scientifici, tesi e paper accademici. Invece di formattare a mano come in Word, scrivi comandi che descrivono la struttura. Il risultato è tipograficamente perfetto, specialmente per le formule matematiche. Curva di apprendimento ripida ma risultati professionali.

\documentclass{article} \begin{document} Il teorema di Pitagora: $a^2 + b^2 = c^2$ \end{document}
Tesi Paper scientifici Matematica Fisica
Overleaf editor online per LaTeX

Overleaf sta a LaTeX come Google Docs sta a Word — è un editor online che ti permette di scrivere LaTeX nel browser, senza installare nulla. Ottimo per collaborare in tempo reale su tesi o paper. Gratuito nella versione base.

LaTeX nel browser Collaborazione Tesi universitarie
💡 Analogia: LaTeX è il linguaggio, Overleaf è il sito dove lo scrivi. Come R e RStudio.
Markdown markup semplice

Versione ultra-semplificata di LaTeX/HTML. Con pochi simboli formatti testo: `**grassetto**`, `# Titolo`, `- lista`. Usato ovunque: GitHub, Notion, questa stessa risposta di Claude è scritta in Markdown.

# Titolo grande **testo in grassetto** - elemento lista - altro elemento
